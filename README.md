## 🚀 How to Run the Programs

### Prerequisites
You need a standard C compiler installed on your system:
* **Linux/macOS:** `gcc` (GNU Compiler Collection)
* **Windows:** MinGW, Code::Blocks, or MSVC

### Compilation Steps
1. Open your terminal or command prompt.
2. Navigate to the directory containing the source files.
3. Compile a specific `.c` program file by typing:
   ```bash
   gcc program_name.c -o program_name
   ```

### Execution
Run the compiled binary executable:
* **On Linux/macOS:**
  ```bash
  ./program_name
  ```
* **On Windows:**
  ```cmd
  program_name.exe
  ```
